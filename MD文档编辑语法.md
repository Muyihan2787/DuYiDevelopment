# MD文档编写基本语法
## 目录
生成目录方式：在文档最顶部写上`[toc]`，在不希望生成目录的内容块其标题后加上`标题{ignore}`则该标题及其下内容块不生成目录。
## 正文【所有用法均需要在一行的开头使用】
1. 标题的几种写法：#……【1-7个】、=====【若干个，表一级标题】、-----【若干个，表二级标题】、#一级标题#【两边(#号个数)级标题】
2. 列表集中写法：无序列表：“*”、“+”、“-”；有序列表：“1.”【数字带英文点加空格】
3. 区块引用（对某部分的内容做一些说明或者引用他人的话）：“>【空格加引用内容】”，“>>>……”重复几次代表几级引用（可以嵌套）
4. 分割线：“*”、“#”、“_”每种符号均可，使用至少三个或以上且后续无内容表示分割线
5. 链接：
- 行内式：”[”+链接名称+]“+“(”+“链接地址”+“"链接的title属性，可缺省"”+“)”
- 参数式：”[”+链接名称+]“+“:”+“链接地址”+“"/'/(链接的title属性，可缺省)/'/"”    【即title参数可用双引号、单引号或小括号】
```md
[门前大桥下](http://home "说明")
[门前大桥下]:http://home "说明"
```
6. 图片：
- 行内式：同上，链接地址放图片地址如https://www.xxx.com/yyy.jpg
- 参数式：在前面多个“!”号
```md
![我是图片](https://www.xxx.com/yyy.jpg)
```
7. 代码：单行用单反引号“`”包裹，多行在开头结尾使用三个单反引号包裹，需单独一行。开头行三个单反引号后跟代码语言，如html
>像这样：```html然后另起一行写代码
8. 强调
- 倾斜强调：用一对“*”或“_”号包裹内容(转为`<em>`效果)
- 加粗强调：用一对“**”或“__”(两个)包裹内容(转为`<strong>`效果)
9. 表格
使用“|”分割单元格，第二行要用格式：“| : ----------- : | : ------- | ----------- : | ”
> 分割线后面的冒号表示对齐方式，写在左边表示左对齐，右边为右对齐，两边都写表示居中
10. 转义：使用“\字符”方式对字符进行转义
11. 删除线：使用一对“~~”符号包裹内容
```md
~~需删除内容~~
```
